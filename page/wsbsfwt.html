    项目介绍

    • 网上办税服务厅：

        按照四川省国税局对网上办税平台开发的基本要求，本系统包括当前税务机关受理涉税事项的主要功能、辅助功能和拓展功能三部分，主要功能包括实现网上登记、纳税申报、发票管理、文书管理、行政救济等基本功能；辅助功能即查询功能，涵盖涉税事项办理状态及结果查询，税收政策及咨询结果查询，税务局通知及提醒以及后期新增的查询；拓展功能包括集成网上发票认证和网络报税等功能。通过网上业务功能的开发整合，为纳税人逐步实现涉税业务一网通办的目标，节约纳税人办税时间，提升征纳双方办税效率，促进税务机关征管与服务职能规范化和标准化目标的实现，和谐征纳关系，提升纳税遵从度。
        系统后端采用Spring + SpringMVC + Mybatis + Shiro + Dubbo + Redis ，前端采用Jquery + jQuery Validate + layer 等插件
